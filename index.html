NLP Based Chatbot

An AI-powered chatbot that uses natural language processing (NLP) to provide accurate responses by correcting spelling errors, preprocessing input text, and returning the most relevant answers.



Description

This project involves developing a chatbot using Flask for web deployment and Streamlit for a user-friendly interface. It leverages TF-IDF vectorization and cosine similarity to deliver responses based on a dataset of predefined questions and answers.

The chatbot processes user input by correcting spelling mistakes, tokenizing text, removing stopwords, and applying lemmatization and stemming. The processed input is then vectorized using TF-IDF and compared with dataset questions using cosine similarity to select the most relevant answer.

Technologies Used
  • Frontend: Flask, Streamlit
  • NLP: NLTK (Tokenization, Lemmatization, Stemming, Stopword Removal)
  • Machine Learning: Scikit-learn (TF-IDF Vectorization, Cosine Similarity)
  • Spelling Correction: PySpellChecker
  • Dataset: CSV file containing predefined questions and answers
Read more
Keywords: Chatbot, Natural Language Processing (NLP), TF-IDF, Cosine Similarity, Flask, Streamlit, Machine Learning

Lung Cancer Prediction Using ML

A web application that utilizes machine learning algorithms to predict the risk of lung cancer based on various genetic and lifestyle factors.



Description

Lung Cancer is a global health concern that requires effective detection methods for successful treatment. Decision tree algorithms is used to handle large datasets and provide accurate predictions. However, another widely used approach for Lung Cancer prediction is logistic regression.

Lung cancer is a leading cause of cancer-related deaths worldwide, necessitating accurate prediction models to improve healthcare outcomes. This study aims to develop a predictive model for lung cancer occurrence by utilizing patient demographics obtained from the National Lung Cancer Institute's Surveillance, Epidemiology, and End Results (SEER) program dataset. The dataset contains various patient features, which will serve as input variables for the logistic regression model.
